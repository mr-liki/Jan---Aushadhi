================================================================================
                         TROUBLESHOOTING
================================================================================

If you get "Permission denied (publickey)":
1. Set up SSH keys: https://docs.github.com/en/authentication/connecting-to-github-with-ssh
2. Or use HTTPS with personal access token instead of password

If you get "fatal: remote origin already exists":
1. Run: git remote remove origin
2. Then: git remote add origin https://github.com/mr-liki/Jan-Aushadhi.git
3. Then: git push -u origin main

If push is slow:
1. Check your internet connection
2. The push may take a few minutes for the first time
3. Be patient and let it complete
